# Mutual Fund Analytics

## Overview

This project is part of the Bluestock Data Analytics Internship.

The objective is to build an ETL pipeline, clean mutual fund data, store it in SQLite, and perform exploratory data analysis.

---

## Technologies Used

- Python
- Pandas
- SQLite
- SQL
- Git & GitHub
